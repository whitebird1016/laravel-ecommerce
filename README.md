# cartwhite

## installation :

- composer install
- cp .env.example .env 
- create new database
- php artisan key:generate
- php artisan storage:link
- php artisan migrate --seed
- npm install
- npm run dev
- php artisan serve

## Permission folder ( Linux | Mac)
- sudo chmod -R 777 storage/app/public/images/
- sudo chmod -R 777 settings.json
